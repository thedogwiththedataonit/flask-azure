Why

Datadog?

Interwoven Monitoring Solution

Azure monitor is interwoven with the underlying application, vm, or service, making aggregate monitoring manual.

Integrations

Azure monitoring is only available for windows and linux systems and geared towards .NET applications.

Synthetics and RUM

Ping tests and multi-step web tests are simplistic and lack the user monitoring capabilities of Datadog.

Proxy Servers

No support for proxy server servies such as Nginx.

Alert Fatigue

Azure handles alert fatigue by aggregating the metric values over reduced granularity. No ML alerting capabilities.

Granularity

Azure's highest granularity is 1 minute.

Log Queries

Azure monitor requires understanding of Kusto Query Language to parse logs.

Superior UI

Azure Monitor's dashboards and UI are nothing special.
